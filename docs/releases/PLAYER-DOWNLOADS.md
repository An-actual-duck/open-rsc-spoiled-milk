# Player Downloads

The limited alpha is intended for invited players connecting to the hosted
Spoiled Milk world. Character progress is saved by that hosted world.

## Downloads

- `spoiled-milk-v0.1.0-alpha.1-windows-x64.zip`: Windows package with a bundled
  Java runtime. Extract it and double-click `Play Spoiled Milk.cmd`.
- `spoiled-milk-v0.1.0-alpha.1-java.zip`: macOS, Linux, or Windows package for
  players who already have Java 17 or newer. Extract it and run
  `./play-spoiled-milk.sh` or `Play Spoiled Milk.cmd`.
- `SHA256SUMS.txt`: checksum file for verifying downloaded archives.

## Alpha Expectations

This release is for controlled playtesting. Game balance, encounter density,
and presentation may change as testing produces useful observations.

Report problems at:
